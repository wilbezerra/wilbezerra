WSB AUTO CLEAN GE

Manual do Usuário atualizado - v3.3.2

Atualização principal: a documentação agora reflete a descoberta inteligente de cache, a proteção reforçada de perfis e a estratégia de release pública com verificação de integridade apenas na versão final disponibilizada ao público.

1. Visão geral

O aplicativo continua operando como utilitário portátil de clique único para ativar/desativar a automação. No boot, executa limpeza inteligente, reconstrução de ícones, sentinela USB e auto recuperação.

AçãoResultado
Primeira execução manualAtiva a automação e registra o estado.
Nova execução manualDesativa a automação e remove a persistência.
Boot com modo ativoExecuta RAM + limpeza + ícones + USB + auto recuperação.

2. Melhorias implementadas

  • Descoberta automática de cache em AppData para aplicativos novos.
  • Whitelist/blacklist para evitar danos em perfis, logins, storage e extensões.
  • Proteção reforçada de Firefox, VS Code e apps Electron/Chromium.
  • Refino da rotina de ícones para fechar apenas a primeira janela automática do Explorer após a reconstrução.
  • Preparação documental da release pública com hash, log de violação e fallback seguro.

3. Preservação de autenticação

Senhas, cookies, tokens persistidos, sessões, Local State e preferências críticas permanecem preservados.

4. Release pública

A proteção de integridade foi reservada somente à versão pública para GitHub. A base de desenvolvimento continua editável.
