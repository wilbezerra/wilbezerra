Os Dois Programas

WSB Auto Clean GE e WSB Flash Clean são programas de limpeza do Windows com propósitos bem diferentes. Um é automático, o outro é manual.

🔵 Auto Clean GE v3.3.2

  • Execução automática ao boot
  • Sentinela USB 24/7 em background
  • Toggle de ativação/desativação
  • Watchdog auto-reconexão (5 min)
  • Para quem quer "set and forget"

🟠 Flash Clean v3.1

  • Execução manual inteligente sob demanda
  • Sentinela USB instantâneo (herdado do AutoClean)
  • Bloqueio anti-execução simultânea
  • Limpeza inteligente preservando logins
  • Motor otimizado (30-60s)
  • Silent mode com Toast + Logs

✓ O que ambos fazem igual

Ambos compartilham a mesma base de limpeza segura, mas com estratégias diferentes. A diferença principal continua sendo QUANDO e COMO: Auto Clean atua de forma automática e contínua; Flash Clean executa sob demanda, agora com motor inteligente herdado do AutoClean.

Arquitetura Inteligente

A partir da versão 3.1, o Flash Clean passou a incorporar partes do motor inteligente do Auto Clean GE v3.3.2.

Isso significa que ambos agora compartilham mecanismos internos de limpeza segura, preservação de sessões, sentinela USB e análise de persistências, mantendo a identidade de cada proposta.

Quando Usar Qual

🔵 Use Auto Clean GE se...

✓ Quer "set and forget" — Ativa uma vez e esquece
✓ Quer proteção contínua — Sentinela 24/7
✓ Conecta pendrives frequentemente — Monitoramento automático
✓ Prefere não pensar em limpeza — Tudo automático
✓ Quer máxima conveniência — Zero intervenção

🟠 Use Flash Clean se...

✓ Quer máximo controle — Você decide QUANDO
✓ Quer execução rápida com inteligência herdada — 30-60 segundos
✓ Não quer background contínuo — Tudo sob demanda
✓ Quer zero impacto no boot — Boot limpo
✓ Prefere simplicidade com segurança extra — Clique & pronto

💡 Dica: Use ambos juntos!

Você pode usar ambos sem conflito. Use Auto Clean para automação contínua e Flash Clean para limpeza rápida sob demanda com a mesma filosofia de segurança e limpeza inteligente.

Fluxo de Operação

Auto Clean GE — Automático ao Boot

1. Windows inicia
Carrega o atalho da pasta Startup automaticamente
2. Aguarda 20s
Deixa o sistema estabilizar antes de começar
3. Executa operações
RAM → Cache → Ícones → Varredura USB (se houver)
4. Sentinela ativa
Monitora pendrives 24/7. Reconecta a cada 5 min se cair

Flash Clean — Manual ao Clicar

1. Você clica
Executa WSB_FLASH_CLEAN.exe e aprova UAC
2. Começa imediatamente
Sem delay. RAM → Cache inteligente → Limpeza de apps → Ícones → Scanner USB → Logs
3. Rápido
Tudo pronto em 30-60 segundos, com bloqueio de clique concorrente durante a execução
4. Encerra
Programa fecha em modo silent. Toast exibido, logs salvos e execução finalizada com segurança
